OriginSix é uma aplicação desenvolvida durante a NLW6, da Rocketseat. Se quiser conhecer mais sobre eles, pode clicar [aqui](https://rocketseat.com.br/). Nesse caso, o website desenvolvido foi o de um salão de beleza.
